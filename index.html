Competitive Programming (2020 - Present)

Competitive Programming (2020 - Present)

I was hooked on solving algorithmic problems ever since I took the Data Structures and Algorithms module offered by NUS. I estimate the number of problems I have solved is probably in the thousands by now. I mainly code on Kattis (top 100 globally) and Codeforces (Expert rank).

As a challenge to myself, I took CS3233 - Competitive Programming, taught by Dr Steven Halim. In this module, students are exposed to high level cp techniques and compete against each other in weekly individual/team contests. It is hands down the most daunting, gruelling and intellectually demanding module I have taken. You can learn more about it here.
